About

This TODOs application is a simple application where a user can write down tasks at need to be completed. Each tasks can either be edited, mark completed, and deleted.

We Respect Your Privacy
This application DOES NOT send any data to any servers. This application implements HTML5's localStorage, meaning any notes created in this application will be stored in the browser's persistent storage.
